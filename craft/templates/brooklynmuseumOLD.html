January 2014

On my first date of hire, there were already three projects set to move. The first was the biggest, the Visitor Experience Initiative, which was as yet undefinded, though the dashboard prototyping was underway. The second was the creation of a micro-site for the event videos for the Elizabeth A. Sackler Center for Feminist Art. And the third, for which I helped define the roadmap, was a refresh of the entire Brooklyn Museum website. Since there was not enough political capital to roll out a complete redesign, we planned to update the header and footer, as well as resetting the type and opening up the layout a little.

  • Began creating a front end style guide for the website.
  • Helped create the RFP for the Sackler Center Video project. Searched for and hired a freelance design/dev team to build it.
  • Comped new headers for the BMA website refresh. A preliminary exercise for selling possible new directions to website stakeholders.
  • Comped new footers for a BMA website refresh.
  • Began designing onboarding process for the VEI dashboard.
